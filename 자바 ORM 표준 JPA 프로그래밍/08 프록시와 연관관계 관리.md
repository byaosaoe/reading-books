# 8.1 프록시
- 엔티티를 조회할 때 연관된 엔티티들이 항상 사용되는 것은 아니다.

- 회원 엔티티
```java
@Entity
public class Member {

  private String username;

  @ManyToOne
  private Team team;

  public Team getTeam() {
    return team;
  }

  public String getUsername() {
    return username;
  }
  ...
}
```

- 팀 엔티티

```java
@Entity
public class Team {

  private String name;

  public String getName() {
    return name;
  }
  ...
}
```
